# Drivelectric Kenya Website Content & Experience Guide

## Brand Theme & UI Direction

### Color Palette

* Primary Background: `#ffffff`
* Alternate Section Background: `#fdfdfd`
* Main Text: `#000000`
* Icon & Accent Color: `#36ac43`
* Footer Background: `#000000`
* Footer Text: `#ffffff`

### Design Style

The website should feel:

* Modern
* Clean
* Sustainable
* Innovative
* Corporate yet futuristic
* Fast and interactive

### Website Experience Goals

* Smooth animations while scrolling
* Interactive cards and hover effects
* Large EV-focused visuals in every section
* Minimal but engaging transitions
* Mobile-first responsive layout
* Fast loading performance
* Easy navigation
